# 壁纸更换功能 - 快速开始

## ✨ 新功能亮点

1. **6种精美预设壁纸** - 从简约到炫酷，总有一款适合你
2. **自定义上传** - 支持任何图片格式
3. **透明度调节** - 精确控制背景显示效果
4. **自动保存** - 设置永久保存，下次打开自动恢复
5. **不影响显示** - 玻璃态设计确保内容始终清晰可读

## 🎯 立即体验

### 步骤 1: 启动项目
```bash
# 后端
cd backend
mvn spring-boot:run

# 前端（新终端）
cd frontend
npm run dev
```

### 步骤 2: 打开浏览器
访问 http://localhost:5173

### 步骤 3: 找到壁纸按钮
在右侧卡片头部，标签页旁边有一个 **🖼️ 图标**

### 步骤 4: 点击并选择
- 点击任意预设壁纸立即应用
- 或上传自己的图片
- 拖动滑块调节透明度

## 🎨 预设壁纸预览

| 名称 | 效果 | 适用场景 |
|------|------|----------|
| 默认网格 | 微妙点状图案 | 日常编程 |
| 深邃星空 | 紫色光晕 | 夜间编程 |
| 极光渐变 | 多彩渐变 | 创意工作 |
| 科技蓝 | 蓝色光晕 | 技术演示 |
| 暖色光晕 | 橙色调 | 温馨氛围 |
| 极简纯色 | 深色背景 | 专注模式 |

## 💡 使用技巧

1. **编程时推荐**: 默认网格 或 极简纯色 + 15% 透明度
2. **演示时推荐**: 科技蓝 或 极光渐变 + 25% 透明度
3. **自定义图片**: 选择暗色系、低对比度的图片效果最佳

## 🔧 技术细节

- **零性能影响**: 纯 CSS 实现，不使用 JavaScript 渲染
- **响应式设计**: 自动适应各种屏幕尺寸
- **本地存储**: 使用 localStorage 保存用户偏好
- **无障碍**: 支持键盘操作和屏幕阅读器
