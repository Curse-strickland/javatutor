# 壁纸文件夹

## 使用说明

将你的壁纸图片放在此文件夹中，支持的格式：
- JPG/JPEG
- PNG
- GIF
- WebP
- SVG

## 推荐尺寸

- **最佳尺寸**: 1920x1080 (16:9)
- **最小尺寸**: 1280x720
- **文件格式**: 建议使用压缩率较高的格式以减小文件大小

## 当前预设壁纸

系统已内置以下壁纸选项：
1. 默认网格 - 微妙的点状网格背景
2. 深邃星空 - 紫色渐变光晕效果
3. 极光渐变 - 多彩渐变背景
4. 科技蓝 - 蓝色光晕效果
5. 暖色光晕 - 橙色温暖色调
6. 极简纯色 - 深色纯色背景

## 添加自定义壁纸

### 方法一：通过界面上传
1. 点击右侧卡片头部的壁纸图标 🖼️
2. 点击 "+ 上传自定义壁纸"
3. 选择你的图片文件

### 方法二：直接放入文件夹
1. 将图片文件复制到此文件夹
2. 在代码中引用（需要修改 WallpaperSelector.vue）

## 注意事项

- 壁纸会自动适应屏幕尺寸（cover模式）
- 支持调节透明度以避免影响内容可读性
- 所有设置会保存在浏览器本地存储中
- 建议选择不干扰代码阅读的暗色系壁纸
