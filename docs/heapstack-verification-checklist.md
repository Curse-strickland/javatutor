## 已知边界行为 (非 Bug)

| 场景 | 行为 | 原因 |
|------|------|------|
| `main(String[] a)` + `Node a = ...` | 编译失败 | Java 变量名冲突，非插桩问题 |
| 空 main body | 0 步骤 | 没有可插桩的语句 |
| 堆 ID 格式 | `0x1A2B` | 基于 `name.hashCode() + heapObjects.size()` 生成，16位 |
| 对象一旦入堆 | 永不删除 | 无 GC，跨步骤保留 |
| 多方法调用 | 仅 main | pushFrame/popFrame 未实现 |
