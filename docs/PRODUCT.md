# Product

## Register

product

## Users

计算机专业大学生。在算法课、数据结构课、或自学时使用。已经学过 Java 基础语法，正在理解排序、递归、动态规划等算法的执行过程。打开浏览器就能写代码、看变量逐行变化，不需要配 IDE。

## Product Purpose

Java 算法可视化教学工具。用户写 Java 代码 → 后端插桩编译执行 → 前端逐步播放变量快照 + 行高亮。核心价值是让抽象的算法执行过程变成看得见的变量变化，降低理解门槛。

## Brand Personality

**专业、清晰、沉浸**。像一本排版精良的教材而非 IDE——足够严肃让人信任教学准确性，但不像 VS Code 那样功能密集带来压迫感。深色主题配合聚焦高亮，让用户把注意力放在代码和变量上，界面主动退后。

## Anti-references

- **Visual Studio Code**: 功能密集的 IDE 布局，工具栏/侧边栏/状态栏争夺注意力。我们的界面应该只有 3 个区域（编辑器、变量、控制栏），没有隐藏菜单。
- **LeetCode**: 白色背景 + 广告 + 付费墙的商业化感。我们免费、纯教学、不卖课。
- **Scratch/Blockly**: 太卡通化，大学生会觉得幼稚。我们的界面面向成年人。

## Design Principles

1. **所见即所得** — 变量值的变化直接显示，不需要用户推理"这行执行后 x 应该是多少"
2. **渐进披露** — 默认只展示核心（代码 + 变量），控制台输出、调用栈等收起，需要时展开
3. **教学准确优先于炫技** — 动画克制（0.2-0.5s），不抢内容；高亮清晰可辨，不止一种颜色
4. **零配置** — 打开浏览器就能用，不需要注册、登录、选择模板

## Accessibility & Inclusion

- WCAG 2.1 AA
- 深色主题默认（教室投影仪环境下可读），后期增加浅色模式
- 动画支持 `prefers-reduced-motion`
- 代码字体至少 14px，变量卡片文字至少 12px
