## 二、共享 Pinia Store（由 F1 提供）

```javascript
// stores/player.js
import { defineStore } from 'pinia'

export const usePlayerStore = defineStore('player', {
  state: () => ({
    steps: [],          // [{ step, line, variables }]
    currentStep: 0,
    isLoading: false,
    error: null,
  }),
  getters: {
    // F2 主要使用这两个
    currentVariables: (state) => state.steps[state.currentStep]?.variables || {},
    currentLine: (state) => state.steps[state.currentStep]?.line || null,
    totalSteps: (state) => state.steps.length,
  },
  actions: {
    async runCode(code) { /* 调用后端 /api/run */ },
    nextStep() { if (this.currentStep < this.totalSteps-1) this.currentStep++ },
    prevStep() { if (this.currentStep > 0) this.currentStep-- },
    reset() { this.currentStep = 0 },
  }
})
F1 承诺：D1 结束前提供 store 初版（含 mock 数据），后续字段变更会提前通知 F2。

## 九、后端接口约定（供参考）

### 1. 执行代码
- **URL**：`POST /api/run`
- **请求**：`{ "code": "public class UserCode { ... }" }`
- **成功响应**：`{ "success": true, "steps": [{ "step": 1, "line": 5, "variables": { "i": 0, "arr": [5,3,8] } }] }`
- **失败响应**：`{ "success": false, "error": "编译错误" }`

### 2. AI 解说（SSE）
- **URL**：`GET /api/explain/{stepId}`
- **响应**：`data: 文本片段\n\n` … 最后 `data: [DONE]\n\n`
