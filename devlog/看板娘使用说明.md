# 高松灯看板娘 — 使用说明

JavaTutor 右下角有一只可爱的企鹅娘陪你写代码、调试、学习 Java。她的名字叫**高松灯**，外号"咕咕嘎嘎"～文静害羞会脸红，但会用心帮你了解每一个功能。

---

## 基础操作

| 操作 | 怎么做 | 会发生什么 |
|:--|:--|:--|
| 看板娘在哪 | 打开 JavaTutor 自动出现 | 右下角，400×400 尺寸，校服小灯 |
| 眼睛跟随 | 移动鼠标即可 | 眼球和头会跟着你的光标转，像在认真注视着你 |
| 摸头 | 点击看板娘的头部 | 随机切换一个表情（惊讶/微笑/思考…） |
| 戳身体 | 点击看板娘的身体 | 播放动作动画 + 害羞地咕咕嘎嘎叫 |
| 快速连戳 | 2 秒内连戳 3/5/7/10 次 | 触发隐藏彩蛋对话（见下方彩蛋表）|
| 隐藏看板娘 | 点击右侧工具栏最下面的 ❌ | 小灯会跟你道别，右下角出现小唤回按钮 |
| 唤回看板娘 | 点击右下角橙色小按钮 | "呜呜，你终于来找我了…" |

---

## 鼠标悬浮提示

把鼠标**悬停**在以下区域，小灯会冒出对应的萌系对话气泡——既可爱又能帮你了解每个区域的功能！

### 代码编辑 & 数据展示区

| 区域 | 功能说明 | 小灯会说什么 |
|:--|:--|:--|
| 代码编辑器 | 写 Java 代码的地方 | "这里是你写代码的地方哦～悄悄看一眼…哇，好厉害的感觉！" |
| 变量展示区 | 当前步骤所有变量的值 | "这里是「变量展示区」～当前步骤所有变量的值都在这里！变化的变量会短暂高亮哦～" |
| 堆 & 栈面板 | 栈帧 + 堆对象可视化 | "栈帧是方法调用的足迹，对象的新家都在堆里～栈帧一层一层像不像千层蛋糕？" |
| 控制台输出 | System.out.println 的结果 | "「控制台输出」～你代码里所有 print 的内容都会出现在这里！" |
| 分屏拖拽条 | 调整编辑器/变量区比例 | "拖这里可以调整编辑器和变量区的大小比例～左右都随你！" |
| 链表可视化 | 链表节点 + 箭头连接图 | "这里是「链表」可视化区！每个节点都是一颗小圆点，用箭头连在一起～" |
| 递归调用栈 | 递归帧的层叠展示 | "「递归调用栈」可视化！每次递归调用都会生成新的栈帧～在这里看得清清楚楚！" |

### 播放控制按钮

| 按钮 | 功能 | 小灯会说什么 |
|:--|:--|:--|
| ▶ 运行 | 编译并执行代码 | "要运行了吗…？我、我有点紧张…(｡ŏ﹏ŏ)" |
| ⏮ 跳到开头 | 跳回第 0 步 | "从头开始看一遍…温故而知新，对吧？" |
| ◀ 上一步 | 后退一步 | "倒退一步…检查刚才变量变对了没有～" |
| ▶ 下一步 | 前进一步 | "前进一小步～离成功又近了一点！٩(ˊᗜˋ*)و" |
| ⏭ 跳到结尾 | 跳到最后一步 | "直接跳到结局！…有点剧透的感觉呢 (･ω<)☆" |
| 自动播放/暂停 | 自动逐步执行 | "自动播放模式～解放双手，专心盯着变量区就好！" |
| 进度条 | 拖动滑块跳到任意步 | "执行到这里了呢！拖动这个蓝色小圆点可以跳到任意一步哦！" |

### AI 解说面板（展开后可见）

点击控制栏上的 ✨ 按钮展开 AI 面板，里面有三个标签页：

| 区域 | 功能 | 小灯会说什么 |
|:--|:--|:--|
| AI 面板整体 | 智能分析代码 | "AI 解说面板～里面有「解说」「复杂度」「算法」三个标签页！" |
| 「解说」标签 | AI 逐行解释当前步骤 | "切换这里可以查看不同类型的 AI 分析内容哦。" |
| 「复杂度」标签 | 时间/空间复杂度卡片 | "这里显示时间复杂度和空间复杂度～是衡量算法好坏的重要指标！" |
| 「算法」标签 | 识别算法类型 + 数据结构标签 | "你的代码用了什么算法和数据结构，都会标在这里哟～" |
| 「解说」按钮 | 手动触发 AI 讲解 | "点一下「解说」按钮，AI 就会解释当前这一步在做什么啦～" |
| 「自动」开关 | 每一步自动解说 | "打开「自动」模式，每一步 AI 都会自动解说～偷懒神器！（小声）" |

## 点击彩蛋 🐧

在 2 秒内快速连戳小灯的身体，触发隐藏彩蛋：

| 连击次数 | 小灯的反应 |
|:--|:--|
| 3 次 | "不要一直戳我啦…虽然不痛但是很害羞的…(\*/ω＼\*)" |
| 5 次 | "咕咕嘎嘎！！…你、你是戳戳星人吗！(╯>□<)╯" |
| 7 次 | "啊啊啊…！不要再戳了啦…代码还在等着你呢…（脸红到耳根）" |
| 10 次 🎉 | "十连击！！你赢了…我认输啦！(oT-T)ノ 咕咕嘎嘎…" ——同时害羞脸红 3 秒 |
| 15+ 次 | 每 5 次一跳："已经第 N 下了…求求你别戳了，再戳我就变成企鹅肉饼了！🐧" |

停手超过 2 秒计数器归零，可以重新挑战。
