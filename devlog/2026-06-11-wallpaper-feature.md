# 壁纸更换功能使用指南

## 🎨 功能概述

本项目现已支持自定义壁纸更换功能，你可以：
- ✅ 从6种预设壁纸中选择
- ✅ 上传自己的图片作为壁纸
- ✅ 调节壁纸透明度
- ✅ 所有设置自动保存，下次打开自动恢复

## 📍 如何使用

### 1. 打开壁纸选择器

在右侧卡片头部（变量/流程/经典标签旁边），点击 **壁纸图标** 🖼️

### 2. 选择预设壁纸

点击任意预设壁纸即可立即应用：
- **默认网格** - 微妙的点状网格背景
- **深邃星空** - 紫色渐变光晕效果
- **极光渐变** - 多彩渐变背景
- **科技蓝** - 蓝色光晕效果
- **暖色光晕** - 橙色温暖色调
- **极简纯色** - 深色纯色背景

### 3. 上传自定义壁纸

1. 点击 "+ 上传自定义壁纸" 按钮
2. 选择你的图片文件（支持 JPG、PNG、GIF、WebP、SVG）
3. 壁纸会自动应用并保存

### 4. 调节透明度

使用底部的滑块调节壁纸透明度（10% - 100%）
- 推荐值：15% - 30%（不影响代码阅读）
- 数值越小，背景越透明

## 💡 最佳实践

### 推荐的壁纸特征
- **暗色系** - 与深色主题更协调
- **低对比度** - 避免干扰代码阅读
- **简洁图案** - 不要太复杂或花哨
- **适当模糊** - 有助于突出前景内容

### 推荐的尺寸
- **最佳**: 1920x1080 (16:9)
- **最小**: 1280x720
- 系统会自动裁剪和缩放以适应屏幕

### 透明度建议
- **编程时**: 15% - 25%（保持专注）
- **演示时**: 30% - 50%（增加视觉效果）
- **纯色背景**: 可以不使用壁纸

## 🛠️ 高级用法

### 使用壁纸生成器

我们提供了一个在线壁纸生成工具：

1. 在浏览器中打开：`frontend/src/assets/wallpapers/wallpaper-generator.html`
2. 调整颜色、类型、透明度等参数
3. 点击"下载壁纸"按钮
4. 将生成的图片上传到应用中

### 添加更多预设壁纸

如果你想添加更多预设选项，编辑 `WallpaperSelector.vue` 文件中的 `wallpapers` 数组：

```javascript
const wallpapers = ref([
  // ... 现有壁纸
  {
    name: '我的新壁纸',
    type: 'gradient',  // 或 'linear', 'solid', 'image'
    value: '你的CSS背景值'
  }
])
```

### 直接使用图片文件

如果你想直接引用文件夹中的图片：

1. 将图片放入 `frontend/src/assets/wallpapers/` 文件夹
2. 在代码中导入：
```javascript
import myWallpaper from '../assets/wallpapers/my-image.jpg'

// 然后在 wallpapers 数组中使用
{
  name: '我的图片',
  type: 'image',
  value: myWallpaper
}
```

## ❓ 常见问题

### Q: 壁纸会影响性能吗？
A: 不会。我们使用了 CSS background 属性，现代浏览器处理非常高效。

### Q: 为什么我的壁纸看起来太亮/太暗？
A: 调整透明度滑块。较亮的壁纸需要更低的透明度（10-20%）。

### Q: 设置会丢失吗？
A: 不会。所有设置都保存在浏览器的 localStorage 中，除非清除浏览器数据。

### Q: 可以在不同设备上同步设置吗？
A: 目前不支持云同步。每个设备的设置是独立的。

### Q: 上传的图片会保存到哪里？
A: 图片会转换为 base64 格式保存在浏览器本地存储中，不会上传到服务器。

## 🎯 设计原则

我们的壁纸系统设计遵循以下原则：

1. **不影响可读性** - 壁纸永远不应该干扰代码阅读
2. **玻璃态保护** - 所有卡片使用 backdrop-filter，确保内容清晰
3. **性能优先** - 使用 CSS 而非 JavaScript 渲染背景
4. **用户控制** - 用户可以完全控制壁纸的显示效果

## 📝 技术实现

- **框架**: Vue 3 Composition API
- **样式**: Scoped CSS + CSS Variables
- **存储**: localStorage
- **图片处理**: FileReader API (base64)
- **动画**: Vue Transition 组件

## 🔧 故障排除

### 壁纸不显示
1. 检查浏览器控制台是否有错误
2. 确认图片格式是否支持
3. 尝试刷新页面

### 透明度调节无效
1. 检查是否在正确的壁纸类型上（纯色背景不支持透明度）
2. 尝试重新选择壁纸

### 自定义壁纸加载慢
1. 压缩图片文件大小（建议 < 500KB）
2. 使用 WebP 格式以获得更好的压缩率
