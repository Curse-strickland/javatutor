## 二、作品目标

**JavaTutor** 是一款面向算法与数据结构学习者的 **Java 代码可视化教学工具**。核心目标是：将抽象的算法执行过程转化为 **清晰可见的变量变化轨迹与内存状态演变**，帮助学习者直观理解程序的每一步在做什么。

用户在浏览器中编写或粘贴 Java 代码，也可通过拖放/上传 .java 文件导入代码。后端通过 AST 插桩、内存编译与安全沙箱执行，将每一步的变量快照、堆/栈内存状态、控制台输出返回前端。前端提供逐步播放器、三色行高亮、变量卡片、堆栈面板、控制流图、AI 流式解说等功能。

### 双重定位

1. **面向初学者**：在学习排序、递归、链表、动态规划等算法时，直观观察数据的变化轨迹，配合 AI 解说理解每一步的含义，降低学习门槛
2. **面向日常编程**：在日常编码过程中，对于复杂逻辑（如多方法调用、嵌套循环、引用传递等）可以快速运行并可视化其执行过程，帮助理解和调试代码

### 界面体验

我们同样注重学习者的编程体验——精心设计的深色主题配合 Maple Mono 等宽字体，提供沉浸式的代码阅读环境。用户可自定义壁纸背景（支持图片、视频上传），搭配背景音乐，让编程学习更加愉悦。整个界面兼顾专业性与美观性，旨在为学习者提供最佳的代码可视化体验。

## 五、使用指南

### 5.1 部署指南

#### 环境要求

| 组件 | 版本要求 |
|------|----------|
| JDK | Java 17 或更高 |
| Maven | 3.6+ |
| Node.js | 18+ |
| npm | 9+ |
| 操作系统 | Windows / macOS / Linux |

#### 后端部署

```bash
# 1. 进入后端目录
cd backend

# 2. 使用 Maven 编译并启动（Spring Boot 内嵌 Tomcat，默认端口 8080）
mvn clean compile
mvn spring-boot:run

# 或者打包为 JAR 后运行
mvn clean package -DskipTests
java -jar target/javatutor-backend-*.jar
```

**配置说明**（`backend/src/main/resources/application.properties`）：

```properties
# 服务器端口
server.port=8080

# 安全沙箱超时（毫秒）
javatutor.sandbox.timeoutMs=5000
```

> **AI 解说配置**：API Key 在前端界面中输入，无需在配置文件中设置。默认使用智谱 GLM-4.7 免费模型，无需任何配置即可使用。

#### 前端部署

```bash
# 1. 进入前端目录
cd frontend

# 2. 安装依赖
npm install

# 3. 启动开发服务器（默认 http://localhost:5173）
npm run dev

# 4. 构建生产版本（输出到 dist/ 目录）
npm run build
```

> **注意**：前端开发模式下 Vite 已配置将 `/api` 请求代理到 `http://localhost:8080`（见 `vite.config.js`），确保后端先启动再打开前端页面。生产部署时可将 `dist/` 静态文件放入后端 `resources/static/` 目录，或使用 Nginx 反向代理。

#### 一键启动脚本

项目提供了便捷脚本：

- **Windows**：双击 `start.bat`（自动启动后端 + 前端）
- **Linux/macOS**：运行 `./start.sh`
- **停止服务**：双击 `stop.bat` 或在终端按 `Ctrl+C`

### 5.2 使用指南

#### 5.2.1 基本使用流程

1. **打开页面**：浏览器访问 `http://localhost:5173`
2. **编写代码**：在左侧编辑器输入 Java 代码（需包含 `public class` 和 `main` 方法）
3. **运行代码**：点击底部控制栏的 ▶（运行）按钮
4. **逐步查看**：
   - 点击 ◀ / ▶（上一步/下一步）单步执行
   - 拖拽进度条跳转到任意步骤
   - 点击 ⏵（自动播放）连续播放，可调节速度
5. **查看变量与流程**：右侧"变量"标签页展示当前步骤的变量卡片和内存状态；"流程"标签页实时显示 Mermaid 控制流图，当前执行步骤的节点会以黄色呼吸灯同步高亮

#### 5.2.2 测试模式

适合运行 LeetCode 风格的代码片段（无需写完整的 `public class` 和 `main`）：

1. 点击编辑器标题栏右侧的"测试"按钮
2. 粘贴 `class Solution { public int[] bubbleSort(int[] arr) { ... } }` 格式的代码
3. 在下方输入测试参数（支持 `int[]`、`String[]`、`List<Integer>`、`ListNode`、`TreeNode` 等类型）
4. 点击运行
5. 查看结果：控制台输出区会显示该函数的返回值（测试模式下 `System.out.println` 被自动替换为返回值收集）

#### 5.2.3 经典代码浏览器

点击右侧"经典"标签页，可浏览 30+ 内置算法示例：

- **排序**：冒泡排序、快速排序、归并排序、插入排序等
- **递归**：斐波那契、阶乘、汉诺塔
- **链表**：反转链表、合并链表、检测环
- **树/图**：DFS、BFS、二叉树遍历
- **动态规划**：背包问题、最长公共子序列

点击任意示例即可加载到编辑器，运行查看可视化过程。

#### 5.2.4 AI 解说

AI 解说面板包含三个标签页：

**「解说」标签页 — 让 AI 讲解代码**
1. 点击控制栏右侧 🐋 按钮展开 AI 面板
2. 点击 **「整体解说」**按钮：AI 综述整段代码的算法思路和数据结构
3. 点击 **「单步解说」**按钮：AI 解释当前这一步在做什么、变量为什么变化
4. 勾选 **「自动」**复选框：步进时自动生成解说，AI 生成中自动暂停等待，生成完毕后继续
5. 已解说的步骤会缓存，回退时从缓存恢复，不重复消耗 token

**「复杂度」标签页 — 自动分析算法效率**
- 运行代码后自动显示时间和空间复杂度（如 O(n²)、O(1)），附带文字说明

**「算法」标签页 — 智能识别算法类型**
- 自动识别代码用到的算法（排序/递归/贪心/DP 等）和数据结构（数组/链表/栈/树等），以彩色标签展示
- 点击任一标签可让 AI 针对该主题深入讲解

**自定义 API — 切换 AI 服务商**
1. 展开面板底部的「自定义 API」折叠区
2. 点击选择平台（智谱 / DeepSeek / OpenAI / Kimi / 自定义）
3. 输入对应 API Key，系统自动校验格式
4. 自定义平台需额外填写 API URL 和模型名称
5. 点击保存 — 仅本次会话生效

> **默认免配置**：不输入 Key 时自动使用内置智谱 GLM-4.7 免费模型，无需任何设置即可使用。

#### 5.2.5 控制流图

1. 点击右侧「流程」标签页，查看由 Mermaid 自动生成的控制流图

3. 点击蓝色方法调用节点可**钻取**进入子方法的流程图，顶部面包屑可返回上层

5. 点击 ⬇ 按钮将当前流程图**下载为 SVG** 文件

3. **自定义背景**：上传图片（JPG/PNG）或视频（MP4）作为背景，视频循环播放
4. **背景音乐**：上传音频（MP3），可调节音量
5. **透明度**：拖动滑块调节卡片透明度（推荐 15%–30%）
